## Common Mistakes / Gotchas

**Submission Mistakes:**
- **Submitting Too Early**: Tool not fully functional or polished
- **Poor Landing Page**: Not optimized for traffic spikes, broken payment systems
- **Unclear Value Prop**: Visitors can't understand what tool does within seconds
- **Missing Visual Content**: No screenshots, videos, or demos hurt performance
- **Generic Positioning**: Trying to be "AI for everything" instead of solving specific tasks

**Platform-Specific Issues:**
- **Pre-Launch on Other Platforms**: Disqualifies from $300 PPC bonus
- **Wrong Package Choice**: Many underestimate value of newsletter feature ($347 vs $49)
- **Ignoring Analytics**: Not monitoring provided data to optimize performance
- **One-and-Done Mentality**: Not using edit functionality to improve listing over time

**Technical Gotchas:**
- Traffic spikes can overwhelm unprepared infrastructure
- Payment processing failures during high-traffic periods
- Broken demo links or signup flows hurt conversion
- Mobile optimization critical (significant mobile traffic)
- International audience requires clear pricing in USD

**Expectation Management:**
- Results vary significantly by product-market fit
- Newsletter feature is one-time, not recurring
- Organic traffic builds over time, initial spike is temporary
- Community expects functional, polished tools not early prototypes
