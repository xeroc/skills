# LaunchIgniter

## Quick Facts
| Metric | Value |
|--------|-------|
| URL | launchigniter.com |
| Registered makers | 4,900+ |
| Backlink | Dofollow DR 74 (Basic/Pro), nofollow (free, except top 3) |
| Cost | Free / $9 Basic / $12 Pro |
| Cadence | Weekly (all launches start Monday 00:00 UTC) |
| Best for | SaaS, AI tools, productivity |

## Submission Requirements
- Product name and tagline (~60 chars)
- Description (benefit-focused, problem/solution)
- Logo (square, clear at 64px)
- Screenshots (2-3 minimum, ideally 4-6)
- Product URL (live, working)
- Category selection

## How to Submit
1. Create account at launchigniter.com
2. Manual submission OR import from Product Hunt / Peerlist (one-click)
3. Choose plan:
   - **Free:** Queue with waiting period, badge verification required
   - **Basic ($9):** Skip queue, instant scheduling, dofollow backlink
   - **Pro ($12):** Skip queue, 5 launch slots per product, dofollow DR 74 + bonus DR 30 from SaaSGrow.app
4. Select launch week (all launches start Monday 00:00 UTC)

## Ranking
- Community upvotes within the weekly window
- Top 3 win Winner badge + permanent dofollow backlink
- Weekly winners archived at launchigniter.com/weekly-winners

## Optimization
- **Pro at $12 is exceptional value:** permanent DR 74 dofollow backlink + 5 re-launch slots
- Import from PH/Peerlist to save setup time
- Engage with other products in your launch week (reciprocity drives votes)
- Premium users get purple gradient border + crown badge (higher click-through)
- Post in r/launchigniter with your launch link
- Plan outreach for Sunday evening / early Monday morning

## Gotchas
- Audience is smaller than PH (don't expect PH-scale traffic)
- Success depends on bringing your own network to vote
- Free tier competes with paid placements in the same weekly batch
- 7-day visibility window is generous (vs PH's 24 hours)
