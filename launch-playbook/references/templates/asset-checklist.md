# Universal Asset Checklist

Prepare assets ONCE for all 39 platforms. This checklist consolidates requirements across Launch Directories, Deal/LTD Marketplaces, and Software Directories.

---

## Essential Assets

### Logo Requirements
- [ ] **Square format** (1:1 ratio, minimum 500x500px)
- [ ] **High resolution** PNG with transparent background
- [ ] **Readable at 40px** (test by shrinking - critical for directory listings)
- [ ] **Works on both light and dark backgrounds**
- [ ] **Strong contrast** - not washed out or hard to see

### Screenshots (3-5 images minimum)
- [ ] **Hero shot:** Core interface doing the main thing
- [ ] **Feature shot 1:** Key feature in action
- [ ] **Feature shot 2:** Second key feature in action  
- [ ] **Results shot:** Outcome/dashboard showing value
- [ ] **Optional 5th shot:** Another feature or use case
- [ ] **High resolution:** 1920x1080px minimum
- [ ] **Real product UI** (not mockups if product is live)
- [ ] **Clean interface** - no personal data, placeholder content used
- [ ] **Device frames added** (use shots.so, screely.com, or tinyshots.app)

### OG Image
- [ ] **1200x630px** (standard social media size)
- [ ] **Shows product in context** - not just logo on gradient
- [ ] **Text readable at small sizes** (will be shrunk for previews)
- [ ] **PNG or JPG format**

### Demo Video (Strongly Recommended)
- [ ] **30-90 seconds long**
- [ ] **Screen recording with founder narration** (authenticity beats polish)
- [ ] **Structure:** Problem (10 sec) → Solution demo (40-60 sec) → CTA (10 sec)
- [ ] **Captions added** (many watch without sound)
- [ ] **Hosted on YouTube or Vimeo** for easy embedding
- [ ] **High quality recording** (1080p minimum)

### Founder Photo  
- [ ] **Professional headshot** (builds trust on community platforms)
- [ ] **Square format** for consistency
- [ ] **High resolution** (500x500px minimum)
- [ ] **Good lighting** and clear background

## Platform-Specific Image Requirements

### Product Hunt
- [ ] **Logo:** 240x240px minimum, square
- [ ] **Gallery:** 4-6 images, 1270x952px or 2400x1260px, PNG, <500KB each
- [ ] **Demo video:** 30-90 seconds, founder narration preferred

### BetaList  
- [ ] **Screenshots:** 1200x750px recommended, 3-5 images
- [ ] **Landing page hero screenshot** IS the preview image

### Peerlist
- [ ] **Personal photo:** Square format (must be founder photo, not logo)
- [ ] **Cover image:** 1200x630px
- [ ] **Fill ALL image slots** for maximum impact

### Hacker News
- [ ] **No images needed** - link to live demo only
- [ ] **Working demo accessible without signup** (critical)

### AppSumo (Deal Marketplace)
- [ ] **Hero/header image:** 1920x1080px minimum, no text overlays
- [ ] **Product images:** Up to 5, maximum 5MB each
- [ ] **Clean screenshots only:** No watermarks, banners, stickers, emojis
- [ ] **Strong logo contrast** with background
- [ ] **Use AppSumo's Figma template** for header formatting

### G2/Capterra (Enterprise Directories)  
- [ ] **Logo:** Square, high-res
- [ ] **Screenshots:** Multiple angles showing enterprise features
- [ ] **Professional presentation** - these are B2B focused

### Developer Platforms (DevHunt, OpenAlternative, SourceForge)
- [ ] **GitHub integration ready**
- [ ] **Code screenshots** or terminal examples
- [ ] **Technical accuracy** in all visuals

### Most Other Directories
- [ ] **Standard web formats:** PNG/JPG
- [ ] **Reasonable file sizes:** Under 5MB typical
- [ ] **2-3 screenshots minimum**
- [ ] **Square logo**

## Copy Assets Needed

### Tagline Variations
- [ ] **60-character version** (Product Hunt limit)
- [ ] **100-character version** (most directories)
- [ ] **Full sentence version** (detailed listings)
- [ ] **Show HN format:** "Show HN: [Name] - [plain English description]"

### Description Blocks
- [ ] **Short:** 1-2 sentences for previews
- [ ] **Medium:** 4-6 sentences for directory pages  
- [ ] **Long:** 300-500 words for comprehensive listings
- [ ] **Technical summary:** For developer-focused platforms

### Special Content
- [ ] **Founder story:** 2-3 paragraphs for community platforms
- [ ] **First comment draft:** For Product Hunt and Hacker News
- [ ] **Maker comment with GIF:** Ready for PH launch day
- [ ] **Technical details:** Stack, architecture for developer platforms
