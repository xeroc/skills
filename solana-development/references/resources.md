## Version Information

**Current versions (as of 2025):**
- Latest Anchor: 0.30+
- Recommended Solana CLI: Latest stable (check with `solana --version`)
- Rust minimum: 1.70+
- Solana program library: 2.0+

**Updating tools:**
```bash
# Update Solana CLI
solana-install update

# Update Anchor
avm install latest
avm use latest

# Update Rust
rustup update
```
